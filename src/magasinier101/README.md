# magasinier101
c'est un POF représentant un cas d'utilisation simple. c'est un magasinier qui esssaye de faire une demande d'achat.

# comment lancer l'application ? 
il suffit de lancer le magasinier101.exe. l'executable se trouve dans ProjetNanoERP\bin\release

# les fonctionnalités :
l'application a deux fonctionnalités principales : 

* insérer une demande d'achat : 
on insère une ligne à la table contenant la liste des demandes
